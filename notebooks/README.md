# Nova Financial Solution — Notebooks

This directory contains Jupyter notebooks for financial data exploration, analysis, and modeling as part of the Nova Financial Solution project.

## Contents

- Notebooks for data preprocessing, exploratory data analysis, and financial modeling.
- Example workflows and experiments.
- Documentation and code comments for reproducibility.

## Usage

1. **Install dependencies:**
    pip install -r ../requirements.txt


2. **Launch Jupyter Notebook:**
    jupyter notebook .

3. **Open and run notebooks:**  
   Follow the instructions within each notebook to explore datasets and models.

## Contribution Guidelines

- Add new notebooks with clear titles and descriptions.
- Comment code and document assumptions or findings.
- Submit pull requests for improvements or new analyses.
